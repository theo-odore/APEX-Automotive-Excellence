# APEX - Automotive Excellence 🏎️

APEX is a premium, high-performance automotive showcase designed with a dark, minimalist aesthetic and fluid animations. It features a dynamic car selection experience with immersive parallax backgrounds and custom performance stats.

🔗 **[View Live Demo](https://theo-odore.github.io/APEX-Automotive-Excellence/)**

![APEX Showcase](https://images.unsplash.com/photo-1544636331-e26879cd4d9b?w=1200&q=80&auto=format&fit=crop)

## ✨ Features

- **Interactive Showcase**: Smooth horizontal navigation through world-class performance vehicles.
- **Dynamic Content**: Each slide updates the UI with tailored brand colors, headlines, and specific performance metrics.
- **Glassmorphism UI**: Modern "frosted glass" stats cards with real-time stat rendering.
- **Immersive Parallax**: Subtle background movement tracking your mouse/cursor.
- **High Performance**: Built with zero external dependencies (Vanilla JS/CSS) for maximum speed.
- **Smooth Loading**: Custom animated loader with percentage tracking.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom animations, transitions, and layout.
- **Vanilla JavaScript**: Logic, state management, and interaction.
- **Google Fonts**: 'Bebas Neue', 'Space Mono', and 'Rajdhani'.

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to view the showcase.
